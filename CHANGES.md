# Внесенные изменения

## 1. Заголовок окна
- **index.html**: Изменен заголовок с "SSH Command Center" на "SSH Config Commander"
- **src-tauri/tauri.conf.json**: Изменен заголовок окна приложения на "SSH Config Commander"

## 2. Кнопка открытия файла конфигурации
- Добавлена кнопка с иконкой `FolderOpen` рядом с полем ввода пути к конфигу
- Кнопка открывает системный диалог выбора файла
- Добавлен плагин `tauri-plugin-dialog` для работы с диалогами:
  - В `src-tauri/Cargo.toml` добавлена зависимость
  - В `src-tauri/src/lib.rs` инициализирован плагин
  - В `package.json` добавлен npm пакет `@tauri-apps/plugin-dialog`
- Добавлена функция `openConfigFile()` для обработки выбора файла

## 3. Удален элемент "Серверы"
- Убран заголовок "Серверы" с иконкой
- Убрана декоративная карточка-обертка
- Все содержимое (форма добавления, таблица серверов) размещено непосредственно в родительском контейнере
- Поле конфигурации теперь первый элемент в интерфейсе

## 4. Кнопка добавления сервера
- Кнопка "Добавить сервер" заменена на иконку плюса без текста
- Добавлена всплывающая подсказка (title) при наведении

## 5. Генерация SSH ключей
- Добавлена кнопка генерации SSH ключа с иконкой `RefreshCw` рядом с кнопкой добавления сервера
- При клике на кнопку генерации:
  - Проверяется существование ключа
  - Если ключ существует - предлагается удалить и создать новый
  - Если ключа нет - создается новый ключ RSA 4096 бит
- Добавлены Rust команды:
  - `generate_ssh_key` - генерация нового SSH ключа
  - `check_ssh_keys_exist` - проверка наличия ключей в ~/.ssh
  - `delete_ssh_key` - удаление существующего ключа

## 6. Проверка ключей при запуске
- При первом запуске приложения проверяется наличие SSH ключей в ~/.ssh
- Если ключи не найдены - показывается модальное окно с предложением сгенерировать новый ключ
- Пользователь может:
  - Сгенерировать ключ сразу
  - Отложить генерацию (кнопка "Позже")

## 7. Дополнительные исправления
- Добавлены TypeScript типы для компонента Server
- Исправлены типы для функций `verifyConnection`, `deleteServer`, `startEditServer`, `saveEdit`
- Добавлены проверки на null для `editingServer`
- Добавлены состояния `showKeyGenerationPrompt` и `isGeneratingKey`
